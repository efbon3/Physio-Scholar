QUESTION 1
Type: recall
Bloom's level: understand
Difficulty (F / I / A): F

Stem: The five phases of the ventricular cardiac action potential are:

Correct answer: Phase 0 (Na⁺ influx — rapid upstroke), Phase 1 (transient outward K⁺ current Ito — early repolarisation), Phase 2 (Ca²⁺ influx through L-type channels balanced by K⁺ efflux — plateau), Phase 3 (delayed rectifier K⁺ — repolarisation), Phase 4 (resting potential set by IK1 inward rectifier)

- "Single phase" — Reveals misconception: this option does not match the canonical answer; review the Explanation for the distinguishing rationale. Correction: see correct answer above for the precise concept.
- "Two phases" — Reveals misconception: this option does not match the canonical answer; review the Explanation for the distinguishing rationale. Correction: see correct answer above for the precise concept.
- "Continuous" — Reveals misconception: this option does not match the canonical answer; review the Explanation for the distinguishing rationale. Correction: see correct answer above for the precise concept.

Explanation: **Ventricular AP phases** [Costanzo ch.4]:

**Phase 0** (~0–5 ms): Voltage-gated Na⁺ channels (Nav1.5) open at threshold (~−65 mV) → rapid inward Na⁺ → membrane reaches +30 mV. Depolarisation rate ~200 V/s. Defines conduction velocity.

**Phase 1** (~5–10 ms): Na⁺ channels inactivate; transient outward K⁺ current (Ito) → brief repolarisation notch.

**Phase 2** (~10–200 ms, the plateau): Inward Ca²⁺ current through L-type Ca²⁺ channels (Cav1.2) balanced by outward delayed rectifier K⁺ currents (IKr — rapid, IKs — slow). Membrane stays near 0 mV. **Triggers CICR** (Ca²⁺-induced Ca²⁺ release) → contraction.

**Phase 3** (~200–300 ms): Cav1.2 inactivates; IKr and IKs continue → rapid repolarisation. IK1 reactivates as Vm becomes more negative.

**Phase 4** (resting): IK1 (inward rectifier, K⁺ outward) sets Vm near EK ≈ −90 mV. In SA/AV node — different (no IK1, spontaneous If).

**Why the plateau is unique to cardiac**:

- Long refractory period prevents tetany.
- Provides Ca²⁺ for CICR.
- Allows controlled, modulable contraction.
- Sets the timescale for diastole (filling).

**Drug pharmacology**:

- Class I: Na⁺ channel blockers (Phase 0).
- Class II: β-blockers (modulate If, Cav1.2 phosphorylation).
- Class III: K⁺ channel blockers (prolong AP — amiodarone, dofetilide).
- Class IV: Ca²⁺ channel blockers (prolong AV nodal AP — verapamil, diltiazem).

Stem: The "funny current" (If) responsible for SA nodal pacemaker activity is carried by:

Correct answer: HCN channels (hyperpolarisation-activated cyclic nucleotide-gated) — predominantly Na⁺ inward current activated by hyperpolarisation; cAMP shifts activation toward more positive voltages → faster pacemaker rate (basis of β-adrenergic chronotropy); ivabradine selectively blocks

Distractors:

- "K⁺ alone" — Reveals misconception: this option does not match the canonical answer; review the Explanation for the distinguishing rationale. Correction: see correct answer above for the precise concept.
- "Ca²⁺" — Reveals misconception: this option does not match the canonical answer; review the Explanation for the distinguishing rationale. Correction: see correct answer above for the precise concept.
- "Cl⁻" — Reveals misconception: this option does not match the canonical answer; review the Explanation for the distinguishing rationale. Correction: see correct answer above for the precise concept.

Explanation: **HCN channels** (HCN4 in SA node):

- Activate at hyperpolarised Vm (~−60 mV onwards).
- Conduct mainly Na⁺ (some K⁺) inward.
- Cyclic AMP binds intracellular site → shifts activation curve to more positive voltages.
- Result: cAMP → faster If → faster diastolic depolarisation → faster pacemaker.

**Pacemaker mechanism in SA node**:

1. After repolarisation, no stable resting potential — slowly drifts toward threshold.
2. **If (Na⁺ inward)** → diastolic depolarisation (Phase 4 in nodal cells).
3. **T-type Ca²⁺ channels** activate at later phase 4.
4. **L-type Ca²⁺ channels** drive Phase 0 (no fast Na⁺ current in nodal cells — slower upstroke).
5. K⁺ currents repolarise.

**Sympathetic acceleration**:

- β1 → cAMP → If activates more → faster diastolic depolarisation → tachycardia.

**Parasympathetic slowing**:

- M2 → Gαi → reduced cAMP → slower If.
- Plus Gβγ → directly opens GIRK (IK,ACh) → hyperpolarisation → slower diastolic depolarisation.

**Ivabradine**:

- Selective If inhibitor.
- Reduces heart rate without negative inotropy or affecting BP.
- Used in chronic heart failure (with reduced EF) when β-blockers insufficient (or contraindicated) for rate control.

**SA node intrinsic rate** ~100 bpm; vagal tone normally slows to ~70 bpm. Cardiac transplant (denervated heart): ~100 bpm.

**Sick sinus syndrome** — Pathological SA node dysfunction; bradyarrhythmias, sinus pauses; treatment with pacemaker.

(Berne & Levy, 7e; Costanzo, 7e — Ch.4.)

Type: recall
Bloom's level: understand
Difficulty (F / I / A): I

Stem: The cardiac conduction system includes (in order of normal conduction):

Correct answer: SA node → atrial musculature + internodal tracts → AV node (with delay ~100 ms) → bundle of His → bundle branches (left and right) → Purkinje fibres → ventricular myocardium

Distractors:

- "SA → ventricles directly" — Reveals misconception: this option does not match the canonical answer; review the Explanation for the distinguishing rationale. Correction: see correct answer above for the precise concept.
- "AV node first" — Reveals misconception: this option does not match the canonical answer; review the Explanation for the distinguishing rationale. Correction: see correct answer above for the precise concept.
- "Purkinje fibres only" — Reveals misconception: this option does not match the canonical answer; review the Explanation for the distinguishing rationale. Correction: see correct answer above for the precise concept.

Explanation: **Normal conduction sequence**:

1. **SA (sinoatrial) node** — Right atrium near SVC. Primary pacemaker (~70 bpm at rest). Spontaneous depolarisation.

2. **Atrial spread** — Through atrial myocardium and specialised internodal tracts (anterior, middle, posterior). Atrial AP duration ~150 ms.

3. **AV (atrioventricular) node** — In interatrial septum near tricuspid valve. **Delay ~100 ms** at AV node — allows ventricular filling. Slow conduction (<0.05 m/s) due to small fibres, fewer gap junctions, and Ca²⁺-driven AP.

4. **Bundle of His** — Through fibrous skeleton (only conduction path between atria and ventricles in normal heart).

5. **Bundle branches** — Right bundle branch (single trunk) + left bundle branch (anterior and posterior fascicles).

6. **Purkinje fibres** — Throughout subendocardium. Very fast conduction (~4 m/s). Allow rapid synchronous ventricular activation (apex first, then base).

7. **Ventricular myocardium** — AP propagates through myocytes via gap junctions. Inside-out (endocardium → epicardium).

**Conduction velocities**:

- Atrial muscle: ~1 m/s.
- AV node: ~0.05 m/s (slowest).
- Bundle of His: ~1.5 m/s.
- Purkinje fibres: ~4 m/s (fastest).
- Ventricular muscle: ~0.5 m/s.

**Pacemaker hierarchy**:

- SA node ~70 bpm.
- AV node ~40–60 bpm.
- Ventricular escape ~30–40 bpm.

**Conduction abnormalities**:

- **AV block**:
  - 1st degree — Prolonged PR (>200 ms); benign.
  - 2nd degree:
    - Mobitz I (Wenckebach) — Progressive PR prolongation then dropped beat. AV nodal. Benign typically.
    - Mobitz II — Constant PR with random dropped beats. Infranodal. Risk of complete heart block. Pacemaker indicated.
  - 3rd degree (complete) — Complete dissociation between atria and ventricles. Pacemaker.
- **Bundle branch block** — RBBB or LBBB. LBBB more often pathological (CAD, cardiomyopathy).
- **WPW (Wolff-Parkinson-White)** — Accessory pathway (Bundle of Kent) bypasses AV node → pre-excitation (delta wave on ECG, short PR). Risk of SVT, AF with rapid ventricular response → VF.

(Berne & Levy, 7e; Guyton & Hall, 14e, Ch.10; Costanzo, 7e.)

Explanation: As detailed in Q1. **Cav1.2 (L-type)** is essential for:

- Plateau maintenance.
- Trigger Ca²⁺ for CICR.
- Modulable target for β-adrenergic enhancement (via PKA phosphorylation).

**Cardiac drugs targeting Cav1.2**:

- **Verapamil, diltiazem** — Cardiac-selective Ca²⁺ channel blockers. Reduce AV conduction (rate control in AF), reduce contractility, reduce coronary vasospasm. Used in HTN, angina, AF rate control.
- **Dihydropyridines** (amlodipine, nifedipine) — More vascular-selective; cause less direct cardiac depression.

**Long QT syndrome** — Genetic mutations in IKr (KCNH2/HERG), IKs (KCNQ1), or other channels → prolonged plateau → torsade de pointes risk. Treatment: β-blockers, ICD if high risk.

**Brugada syndrome** — SCN5A loss-of-function → reduced INa → ST elevation V1-V3 → risk of polymorphic VT/VF → ICD if symptomatic or strong family history.

(Boron & Boulpaep, 2e — Ch.21; Costanzo, 7e — Ch.4.)

Stem: The action potential of the AV node differs from ventricular muscle in that it:

Correct answer: Has a slow Phase 0 driven by L-type Ca²⁺ channels (no fast Na⁺ channel-mediated upstroke); slow conduction velocity (~0.05 m/s vs ~0.5 m/s ventricular); explains the AV nodal delay; long refractory period limits conduction during AF

Distractors:

- "Is identical" — Reveals misconception: this option does not match the canonical answer; review the Explanation for the distinguishing rationale. Correction: see correct answer above for the precise concept.
- "Has no Phase 0" — Reveals misconception: this option does not match the canonical answer; review the Explanation for the distinguishing rationale. Correction: see correct answer above for the precise concept.
- "Is faster" — Reveals misconception: this option does not match the canonical answer; review the Explanation for the distinguishing rationale. Correction: see correct answer above for the precise concept.

Explanation: **AV nodal cells**:

- Slow upstroke (Phase 0) — Cav1.2-driven (no Na⁺ channels).
- Slow conduction.
- Long refractory period.
- Spontaneous depolarisation (escape rhythm 40–60 bpm).

**Functional consequences**:

- **AV delay** (~100 ms) — Allows ventricular filling before contraction.
- **Rate-limiting in AF** — AV node cannot conduct every atrial impulse (which can be 400-600 bpm in AF) → ventricular rate ~120–180 bpm.
- **Drugs targeting AV node** for rate control:
  - **β-blockers** — Slow Cav1.2 phosphorylation.
  - **Calcium channel blockers** (verapamil, diltiazem) — Direct Cav1.2 block.
  - **Digoxin** — Indirect via vagal stimulation.
  - **Adenosine** — Activates A1 → Gi → reduces cAMP → slows AV conduction. Used to terminate SVT.

**Vagal manoeuvres** (carotid sinus massage, Valsalva) → increase vagal tone → slow AV node → can terminate AVnRT/AVRT.

**Cardiac glycosides (digoxin)**:

- Inhibit Na⁺/K⁺-ATPase → raised intracellular Na⁺ → reduced NCX function → raised intracellular Ca²⁺ → positive inotropy.
- Plus increased vagal tone → AV nodal block (rate control).
- Narrow therapeutic window. Toxicity: arrhythmias (atrial tachycardia with AV block, ventricular arrhythmias), nausea, yellow-green visual halos, confusion. Hypokalaemia worsens. Treatment: stop drug, K⁺ correction, digoxin-binding antibodies (digifab) if life-threatening.

(Boron & Boulpaep, 2e — Ch.21.)
